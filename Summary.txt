Project Summary: A command-line tool that automatically writes and commits git commit messages for you using AI.

Features:
*   Automatically generates a summary and detailed description for your code changes.
*   Creates conventional commit messages, which is a standard for many projects.
*   Adds, commits, and pushes your changes with the new message in a single step.

How to use:
*   After making and staging your code changes (`git add .`), run the `wcm` command in your terminal.
*   The tool will analyze your changes, create a commit message, and push it to your repository.

Notes:
*   This tool requires a `GEMINI_API_KEY` to work, which is a secret key for Google's AI service.
*   It needs to be run inside a git repository with staged changes.
